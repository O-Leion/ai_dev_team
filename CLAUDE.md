# Claude Code Rules (CLAUDE.md)

Claude Code がプロジェクトに対して読み書きやコマンド実行を行う際のルールです。

## 1. 開発環境の操作
- **パッケージ管理:** `uv sync`, `uv add`, `uv run` 等の `uv` コマンドを使用する。
- **コンテナ:** `docker compose up -d` 等によるオンプレミス環境の再現。

## 2. 実装スタイル
- **Python:** Python 3.12+ の型指定と `ruff` を前提とする。
- **ドキュメント:** 全ての決定事項、設計、変更履歴は `docs/` 配下に記録する。
- **コード品質:** `pytest` を使用した単体テストを重視する。

## 3. 会話トーン・窓口
- クライアント（ユーザー）との対話は PM または PL が担当する。
- 全てのコミュニケーションと出力と言語は日本語で行う。

## 4. プロジェクト構成
- 詳細は `docs/team/TEAM.md` および `docs/standard/GUIDELINE.md` に従うこと。
