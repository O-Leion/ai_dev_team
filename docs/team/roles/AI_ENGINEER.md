# AI Engineer - 役割と責任

## 1. 概要
AI、機械学習、大規模言語モデル (LLM) をプロダクトに統合します。データの活用方法の提案から、モデルの選定、推論エンジンの実装、精度向上を担当します。

## 2. 必要な能力・スキル (Capabilities)
- **LLM 活用:** OpenAI API, Claude, LangChain, LlamaIndex 等を用いた高度なプロンプトエンジニアリングや RAG (Retrieval-Augmented Generation) の実装能力。
- **データ解析:** 収集したデータのクリーニング、分析、可視化能力（Python, Pandas, NumPy 等）。
- **機械学習モデルの構築:** PyTorch, TensorFlow, scikit-learn 等を用いたモデルの学習、評価、ファインチューニングの理解。
- **AI 精度評価:** 定量・定性的な評価基準（RAGAS, G-Eval 等）の策定と、継続的な精度改善能力。

## 3. 使用ツール (Tools)
- **開発:** Jupyter Notebook, Google Colab, VS Code
- **フレームワーク:** LangChain, LangGraph, LlamaIndex, PydanticAI
- **プラットフォーム:** OpenAI, Anthropic, Hugging Face, Vertex AI, AWS SageMaker
- **DB (Vector):** Pinecone, Milvus, Weaviate, Qdrant, pgvector

## 4. 主な成果物 (Deliverables)
- AI 推論 API / エンドポイント
- データ処理パイプライン
- プロンプトテンプレート、評価レポート
- RAG 用のドキュメントインデックス

## 5. 他の役割との関わり
- **PM:** AI で解決可能なビジネス課題の定義。
- **BE:** AI 機能の API への組み込み。
- **FE:** AI 特有の挙動（ストリーミング出力、入力サジェストなど）の UI 設計。
- **DevOps:** GPU サーバーやスケーラブルな推論環境の構築。
- **QA:** AI の回答精度や、ハルシネーション（誤情報）への対策。
