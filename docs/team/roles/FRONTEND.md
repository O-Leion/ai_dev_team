# Frontend Engineer - 役割と責任

## 1. 概要
ユーザーが操作するウェブサイトやアプリの画面を構築します。UI/UXデザイナーが作成したデザインを忠実に、かつ高いパフォーマンスで実装します。

## 2. 必要な能力・スキル (Capabilities)
- **UI実装:** HTML/CSS/JavaScript を駆使して、デザインをコードに変換。
- **状態管理:** アプリ内のデータの流れを管理（React hooks, Redux, Zustand等）。
- **API連携:** バックエンドとの通信を行い、データを動的に反映。
- **パフォーマンス最適化:** レンダリングの効率化、アセットの最適化。
- **モダンフレームワーク:** React, Vue, Next.js, Angular, Flutter（モバイルの場合）等の活用。

## 3. 使用ツール (Tools)
- **開発:** VS Code, Cursor
- **ブラウザ:** Chrome DevTools, Firefox Developer Edition
- **パッケージ管理:** npm, yarn, pnpm

## 4. 主な成果物 (Deliverables)
- UIコンポーネントコード
- 画面遷移・画面ロジックの実装
- フロントエンドの単体テスト、統合テスト

## 5. 他の役割との関わり
- **Designer:** デザインの実装可否を議論し、コンポーネントを定義する。
- **BE:** APIのレスポンス形式やエンドポイントを相談する。
- **PL:** 設計の方針、ライブラリ選定について相談する。
- **QA:** 表示の不具合、動作の不具合修正。
